# Hide Desktop Icon Labels

A [Windhawk](https://windhawk.net/) mod that removes the text labels from Windows desktop icons and replaces the selection highlight with a clean square centered on the icon.

## What It Does

- Hides all icon label text on the desktop
- Replaces the default rectangular highlight (which covers the icon and label) with a compact square that wraps just the icon
- Automatically adapts to your current desktop icon size (small, medium, large, extra large)

## Before / After

| Default | With Mod |
|---|---|
| Icon with "Recycle Bin" text and tall highlight | Icon only with square highlight, no text |

## Installation

1. Install [Windhawk](https://windhawk.net/)
2. Open Windhawk and go to **Explore Mods** or create a new local mod
3. Paste the mod source code
4. Enable the mod — changes take effect immediately, no restart needed

## Compatibility

- Windows 10 / Windows 11
- x86-64 only

## Uninstalling

Disable or remove the mod in Windhawk. The desktop will immediately repaint with labels restored.
